## Git Commit & PR Message

| 태그이름    | 설명                                                  |
| ----------- | ----------------------------------------------------- |
| ✨ Feat     | 새로운 기능 추가                                      |
| ⬆️ Update   | 수정 추가                                             |
| 🐛 Fix      | git과 관련된 버그 수정                                |
| 🎨 Design   | CSS 등 UI 수정                                        |
| ✏️ Style    | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우 |
| ♻️ Refactor | 코드 리팩토링                                         |
| 📝 Docs     | 문서 수정 (MD 파일)                                   |
| ✅ Test     | 테스트 코드를 생성하거나 수정 하는 경우               |
